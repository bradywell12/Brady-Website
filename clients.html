Import LinkedIn Connections

Supported formats — upload any of these:

  1. iPhone / iCloud Contacts (.vcf): Go to iCloud.com → Contacts → gear icon → Select All → Export vCard → upload the .vcf file directly here
  2. LinkedIn (.csv): Settings & Privacy → Data Privacy → Get a copy of your data → Connections → download Connections.csv
  3. Google Contacts (.csv): contacts.google.com → Export → Google CSV format
  4. Custom spreadsheet (.csv): Any CSV with First Name, Last Name, Phone, Email columns
Tip: You can upload .vcf or .csv — both work. Phone, email, and name are all imported automatically.
